## Description
Backend application for organizing sports matches. Users can create matches, join games, and manage participants.

## Tech Stack
- FastAPI
- PostgreSQL

## Features
- Create match
- Join match
- View players
- Authentication

## Architecture
- routers/
- models/
- database/
